# 블로그 만들기
---
## 프로젝트 소개
### Spring Boot를 이용하여 자유롭게 게시글을 쓸 수 있는 웹사이트를 구축
---
## 개발 환경
* MySQL
* Apache Tomcat
* JAVA(JDK 1.8)
* Spring Boot
* STS4
* Spring Security
* Summernote
* Bootstrap4
* Kakao Login API
---
## 주요 기능
* 로그인 및 회원가입: 일반 유저, 카카오 API
* 게시글: 작성, 수정, 삭제
* 댓글: 작성, 삭제
* 회원정보: 회원정보 수정
---
